# Back To The Future - Time Machine

## How to build

mvn clean install
cd target
java -jar timemachine-1.0.0.jar

## How to play

Keys:

1 - Turn on

2 - Turn off

arrows - select number

+/- - change value

8 - Accelerate

9 - Decelerate
